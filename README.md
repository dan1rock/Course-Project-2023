## How to use
The project requires node.js to be installed on your machine. It was originally built on v20.10.0

Install all dependencies:
```bash
$ npm install
```

To run application:
```bash
$ npm start
```
Open http://localhost:3000 to view it in your browser.

To run tests:
```bash
$ npm test
```
